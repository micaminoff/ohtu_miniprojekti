CREATE TABLE Vinkki (
  id integer NOT NULL PRIMARY KEY AUTOINCREMENT,
  type varchar(15) NOT NULL,
  comment varchar(160),
  tags varchar(160),
  related_courses varchar(160),
  esitietokurssit varchar(160)
);

CREATE TABLE Book (
  id int NOT NULL,
  author varchar(100) NOT NULL,
  title varchar(100) NOT NULL,
  ISBN varchar(30),
  FOREIGN KEY (id) REFERENCES Vinkki(id)
);

Ideana on siis, että kun lisätään uusi vinkki, sille lisätään automaattisesti id (AutoIncrement).
Ohjelmassa sitten kaivetaan tuo äsken lisätty id, ja lisätään tämä sama id tyypin mukaiselle
taululle. Eli esim, lisätään vinkki, se saa id:ksi 3 ja tyypiksi book. Nyt kaivetaan tuo 3 ohjelmalla
ja lisätään tauluun Book olio, jolle annetaan id:ksi 3.
Nyt sit voidaan vaikka SELECT * FROM Vinkki v, Book b WHERE v.id = b.id, niin saadaan listattua
sekä vinkin, että kirjan tiedot.
